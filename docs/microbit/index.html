Micro:Bits

  • First, install this driver for Windows.
  • Download Mu. You will use this to program the Micro:Bit.
  • Pick up a Micro:Bit, plug it in to your computer, and start Mu.
  • Click the REPL button. Try typing display.show(Image.HAPPY) . If it worked you should see an image on the screen.
  • Try some of the things in this tutorial.
  • Come up with something interesting to do with the Micro:Bit. Besides the screen and buttons, it has an accelerometer (it can detect when you tilt it), a compass, a radio (it can communicate with other Micro:Bits), and 3 pins at the bottom (you can connect these to different electronic devices like a speaker, or it you can press them with your fingers). You can find the complete list of Python functions for the Micro:Bit in the API documentation.
  • Pick up a robot if one is available. If you connect a Micro:Bit to the robot, it will be powered by the batteries and it has a speaker and 2 motors that it can control. Download this file. This is the base that will allow you to control the motors—you can add your own code to the bottom. You can spin the motors with left_motor.run(speed) or right_motor.run(speed) where speed is a number from -1 (full speed reverse) to 1 (full speed forward).
