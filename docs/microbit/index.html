Micro:Bits

  • First, install this driver for Windows.
  • Download Mu
  • Pick up a Micro:Bit, plug it in to your computer, and open Mu.
  • Click the REPL button. Try typing display.show(Image.HAPPY). If it worked you should see an image on the Micro:Bit
  • Try some of the things in this tutorial.
  • Come up with something interesting to do with the Micro:Bit. Besides the screen and buttons, it has an accelerometer (it can detect when you tilt it), a radio (it can communicate with other Micro:Bits), and 3 pins at the bottom (you can connect these to different electronic devices like a speaker, or it you can press them with your fingers). You can find the complete list of Python functions for the Micro:Bit in the API documentation.
