Using the Seamonsters Library

The Seamonsters Python library is a collection of useful code for programming robots, based on what we wrote last year. This includes utilities for getting input from joysticks and gamepads, driving with different types of drivetrains, and simulating the robot on your computer without an actual robot, as well as tools to help organize robot code.

Currently the library is kept in the SeamonstersTemplate repository. The code is here, and the documentation is here.

In order to use it, you will fork the SeamonstersTemplate repository. This lets you have your own copy of the repository on your own account, where you can make whatever changes you want, and still be able to get updates to the library. Go to the library on GitHub and click "Fork" in the top-right corner.

Now you can clone your own fork of the repository to your computer. Open Git Bash (or a Terminal on Mac/Linux) and type:

cd Documents
git clone https://github.com/Seamonsters-2605/SeamonstersTemplate.git

You will now have a folder in your Documents folder called “SeamonstersTemplate.” This has the Seamonsters code library in the “seamonsters” folder. To use this library, you just need to have your robot file in the SeamonstersTemplate folder.

Gamepads

One useful thing the Seamonster library adds is support for gamepads. We normally use two Logitech gamepads for driving the robot in competitions. Wpilib treats gamepads as a single “joystick,” but they actually have 2 joysticks on them, which makes them confusing to work with. To make things easier Seamonsters library has a Gamepad class which adds extra features to the wpilib.Joystick class, such as distinguishing between the left and right joystick.

Try this code. It’s a variation on the first tank drive code you wrote earlier.

__author__ = "Your name here!"

import wpilib
import seamonsters.gamepad

class TestRobot (wpilib.IterativeRobot):

  def robotInit(self):
    self.leftFront = wpilib.CANTalon(2)
    self.rightFront = wpilib.CANTalon(1)
    self.leftBack = wpilib.CANTalon(0)
    self.rightBack = wpilib.CANTalon(3)

    # Instead of 2 joysticks, you just need one Gamepad, which has 2
    # joysticks on it.
    self.gamepad = seamonsters.gamepad.Gamepad(0)

  def teleopPeriodic(self):
    # Get the y position for the left and right joysticks on the
    # Gamepad
    leftSpeed = -self.gamepad.getLY()
    rightSpeed = self.gamepad.getRY()

    self.leftFront.set(leftSpeed)
    self.leftBack.set(leftSpeed)
    self.rightFront.set(rightSpeed)
    self.rightBack.set(rightSpeed)

if __name__ == "__main__":
  wpilib.run(TestRobot)
