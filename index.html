Seamonsters Robotics Control Systems Training/Reference

(FRC Team 2605)

Most of the materials on this site are meant for use by the Seamonsters control systems group, but other teams may find some of it useful. If you are looking for the official Seamonsters team site, you should go to sehomeseamonsters.org instead.

The Seamonsters Control Systems group is for electronics and programming for the robot. We use Python with pyfrc for writing robot code.

Resources

  • Codecademy: there are good tutorials on here for Python and Git. The Git tutorial is quick and easy to follow. The Python tutorial is a bit longer (you can skip Units 9 and 10 and stop after 11). It may be slow for you if you have already done some programming in another language—in that case, you should try the python.org tutorial below.
  • The Python Tutorial: This is recommended if you already know the basic concepts of programming, or you need to refresh yourself on Python syntax. Make sure you look at Part 9 (Classes).
  • RobotPy documentation: library for controlling the robot.
  • robotpy-wpilib-utilities documentation: Extra utilities for RobotPy.
